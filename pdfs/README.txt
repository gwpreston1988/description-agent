PDF Spec Sheets Directory
==========================

Place all HVAC product specification PDF files in this directory.

The agent will automatically search for PDFs by model number.

Example filenames:
- SS-GLZS4B-R32.pdf (Heat Pump specs)
- SS-AMST24BU13.pdf (Air Handler specs)
- SS-GR9S.pdf (Furnace specs)
- SS-CAPEA.pdf (Coil specs)

The agent supports flat structure or subdirectories - it will search recursively.
